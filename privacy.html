Privacy Policy

Tilt Battle Royale (last updated 2026)

We collect the minimum data needed to run multiplayer games on Discord:

  • Discord user ID, username, and display name when you join a match or open the spectator view
  • Channel ID to route live game updates to the correct spectator session
  • Game state (party roster, HP, events) held in memory for the duration of a match

We do not sell your data. Match data is not persisted to a database after the game ends unless we add optional stats features later (with an updated policy).

Discord’s own privacy policy applies to your use of Discord: discord.com/privacy
